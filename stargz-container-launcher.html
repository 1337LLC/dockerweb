How It Works

🎯 Lazy Loading Benefits
  • Instant Start: Containers begin running immediately
  • Bandwidth Efficient: Only download what you use
  • Scalable: No image modification needed
  • Browser Native: Uses standard HTTP range requests

Example Commands

# List system information
uname -a

# Check available packages (Alpine)
apk info

# Run Python (if Python image)
python3 --version
python3 -c "print('Hello from container!')"

# Download a file
wget https://raw.githubusercontent.com/stargz-containers/stargz-containers/main/README.md

# Run a web server
python3 -m http.server 8000

# Check disk usage
df -h
du -sh /
